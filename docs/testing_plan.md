# План тестирования

## 1. Цели и задачи тестирования
Тестирование проводится с целью проверки логики работы отдельных частей проекта.  
**Задача:** Проверить корректность математических вычислений и обработки граничных значений

## 2. Описание тестируемого продукта
Проект вычисляет площадь и периметр для таких фигур как: квадрат, круг, используя стандартные математические формулы    

## 3. Область тестирования
В ходе тестирования будут исследованы следующие файлы:
- `circle.py` 
  - `perimetr(r)`
  - `area(r)`
- `square.py`
  - `perimetr(r)`
  - `area(r)`

## 6. Ожидаемые результаты
По результатам тестирования будет подготовлен отчет, содержащий: 
1) Количество выполненных тестов и процент успешных
2) Список обнаруженных дефектов с приоритетами
3) Рекомендации по доработке
